##Proyecto de envío de resumenes

1. Mecánica
Un sistema sencillo hecho con Laravel en su versión 10, utilizado para la recopilación de información de los clientes y sus requerimientos respecto a trabajos que soliciten

2. Diseño
Se utilizó Bootstrap para todo lo que respecta a diagramación y CSS, facilitando el proceso de estilización

3. Base de datos
La conexión la hace con la base de datos del sistema CRM de la agencia para la inserción, edición y eliminación de datos
